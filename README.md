# 🌤️ Weather App

## 📌 Project Overview
The **Weather App** is a Python-based application that provides real-time weather information for any city. The application fetches weather data from an external weather API and displays important details such as temperature, humidity, weather conditions, and wind speed.

The main objective of this project is to demonstrate how to integrate **APIs with Python** to retrieve and display real-time data. This project also helps in understanding API requests, JSON data handling, and user interaction in Python applications.

---

## 🚀 Features
- Get real-time weather information
- Search weather by city name
- Displays temperature, humidity, and weather conditions
- Fetches live data using a weather API
- Simple and user-friendly interface

---

## 🛠️ Technologies Used
- **Python**
- **Requests Library**
- **Weather API**
- **JSON Data Handling**
